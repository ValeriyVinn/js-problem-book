Welcome!

Web-technologies

Websites are the backbone of the internet, and they are all built on three main technologies.

  • HTML - is responsible for filling the web page with content, structure and content.
  • CSS - used for design and positioning, appearance.
  • JavaScript - allows you to add functionality, reaction to user actions.

Веб-технології

Веб-сайти - це основа, на якій тримається інтернет, і всі вони побудовані на трьох основних технологіях.

  • HTML - відповідає за наповнення веб-сторінки контентом, структуру і зміст.
  • CSS - використовується для оформлення та позиціонування, зовнішній вигляд.
  • JavaScript - дозволяє додати функціонал, реакцію на дії користувача.

Для того щоб створити веб-сайт, недостатньо тільки цих базових технологій. Під час розробки використовуються редактори коду, системи контролю версій, збирачі, фреймворки, бібліотеки, хостинг та інші концепції, з якими необхідно буде познайомитися.

In order to create a website, only these basic technologies are not enough. During development, code editors, version control systems, assemblers, frameworks, libraries, hosting and other concepts are used, with which it is necessary to become familiar.

HTML - Hypertext Markup Language

HyperText Markup Language a web document markup language. A set of rules for structuring (marking) text information, adding images, creating tables, forms, lists, etc.

HTML-документ - text file with the extension .html . A document marked up using HTML is interpreted by the browser, as a result of which users see not the source code with markup elements, but the final result of processing - a web page.

HTML - мова розмітки гіпертексту

HyperText Markup Language (мова розмітки гіпертексту) - мова розмітки веб-документів. Набір правил для структурування (розмітки) текстової інформації, додавання зображень, створення таблиць, форм, списків тощо.

HTML-документ - текстовий файл з розширенням .html . Документ, розмічений за допомогою HTML, інтерпретується браузером, в результаті чого користувачі бачать не вихідний код з елементами розмітки, а остаточний результат обробки - веб-сторінку.

Популярність HTML і використання його як основи веб-сторінок, призвело до необхідності створення, підтримки і розвитку стандартів. Зараз цим займається Web Hypertext Application Technology Working Group (WHATWG) , яка працює над специфікацією HTML Living Standard.

The popularity of HTML and its use as the basis of web pages has led to the need to create, maintain and develop standards. This is currently being handled by the Web Hypertext Application Technology Working Group (WHATWG) , which is working on the HTML Living Standard.
