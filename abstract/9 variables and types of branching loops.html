Introduction to JavaScript

When we talk about programming, the first thing that comes to mind is a set of instructions in a file. Source code (source code) is the text of a computer program in any programming language, a set of easy-to-read syntactic structures that describe a set of instructions for a computer.


Actually, the source code itself is incomprehensible to the machine. There is a step that takes place after the program is written that converts the source code in the file into a set of instructions that the computer can understand. This is done by a special program: a compiler or an interpreter. From this we can draw a conclusion - the code is not written for machines, but for the developer. The source code must not only correctly solve the problem, but also be understandable.

Знайомство з JavaScript

Коли ми говоримо про програмування, перше, що спадає на думку - це набір інструкцій у файлі. Вихідний код (source code) - це текст комп'ютерної програми будь-якою мовою програмування, набір легких для читання синтаксичних конструкцій, що описують набір інструкцій для комп'ютера.


Власне сам вихідний код незрозумілий для машини. Існує крок, що виконується після написання програми, який конвертує вихідний код у файлі в набір інструкцій, зрозумілих комп'ютеру. Цим займається спеціальна програма: компілятор або інтерпретатор. З цього можна зробити висновок - код пишеться не для машин, а для розробника. Вихідний код повинен не лише правильно вирішувати задачу, але й бути зрозумілим.

Logical thinking

There are many programming languages in the world. They are not as complex as human ones, because they consist of a fairly small set of syntactic structures, and their working principles and basic concepts are similar between different languages.


Experienced developers look at problems from the point of view of algorithms - a set of steps that must be performed to achieve a certain goal. We use algorithms all the time in our daily lives. For example, the process of making tea is a set of necessary steps (algorithm) to achieve a result. When you get used to building a solution to a problem in the form of an algorithm, the programming language will be just a tool.

Логічне мислення​

В світі існує багато мов програмування. Вони не такі складні, як людські, тому що складаються з досить маленького набору синтаксичних конструкцій, а їх принципи роботи та базові концепції схожі між різними мовами.


Досвідчені розробники розглядають проблеми з точки зору алгоритмів - набору кроків, які потрібно виконати для досягнення певної мети. Ми постійно використовуємо алгоритми у повсякденному житті. Наприклад, процес приготування чаю - це набір необхідних кроків (алгоритм) для досягнення результату. Коли ви звикнете вибудовувати рішення задачі у вигляді алгоритму, мова програмування буде всього лиш інструментом.

JavaScript

JavaScript is an implementation of the EcmaScript specification, a high-level programming language supported by all modern web browsers. First of all, it is intended for interaction with elements of web pages and adding interactivity.

JavaScript - реалізація специфікації EcmaScript, високорівнева мова програмування, яка підтримується усіма сучасними веб-браузерами. В першу чергу призначена для взаємодії з елементами веб-сторінок і додавання інтерактивності.

! INTERESTING
JavaScript is in no way related to the Java language. These are two independent programming languages.
! ЦІКАВО
JavaScript жодним чином не стосується мови Java. Це дві абсолютно незалежні мови програмування.

In front-end development, JavaScript is used together with HTML and CSS to provide the functionality of a web page, namely:

  • simple calculations
  • verification and manipulation of data entered by the user
  • storing information in the user's browser
  • dynamic change of the HTML document
  • reaction to user actions
  • creation of interactive elements: galleries, graphs, etc.
  • interaction with the backend

У Front-end розробці, JavaScript використовується разом із HTML і CSS для забезпечення функціональності веб-сторінки, а саме:

  • нескладні обчислення
  • перевірка і маніпуляція даними, введеними користувачем
  • зберігання інформації в браузері користувача
  • динамічна зміна HTML-документа
  • реакція на дії користувача
  • створення інтерактивних елементів: галерей, графіків і т. п.
  • взаємодія з бекендом
! INTERESTING

If the JavaScript code is written to be executed in the browser (Front-end), the developer does not have tools and access to the file or operating system for security reasons.

! ЦІКАВО

Якщо JavaScript-код пишеться для виконання в браузері (Front-end), у розробника відсутні інструменти і доступ до файлової або операційної системи з міркувань безпеки.

Today, using JavaScript, you can create:

  • web applications on React, Vue, Angular and other frameworks
  • backend applications on Node.js
  • mobile applications on React Native or Ionic
  • desktop applications using Electron
  • microcontrollers with Johnny-Five and Espruino

На сьогодні, використовуючи JavaScript, можна створювати:

  • веб-застосунки на фреймворках React, Vue, Angular та інших
  • бекенд-застосунки на Node.js
  • мобільні застосунки на React Native або Ionic
  • десктоп-застосунки за допомогою Electron
  • мікроконтролери з Johnny-Five і Espruino

To realize all these possibilities, it is necessary to know well the syntax of the language, its features and mechanisms, train to think algorithmically and solve as many practical problems as possible.

Для реалізації всіх цих можливостей, необхідно добре знати синтаксис мови, її особливості та механізми, тренуватися мислити алгоритмічно і вирішувати якомога більше практичних задач.
