Task 1

Завдання

HTML містить список категорій ul#categories.

  <ul id="categories">
    <li class="item">
      <h2>Animals</h2>
      <ul>
        <li>Cat</li>
        <li>Hamster</li>
        <li>Horse</li>
        <li>Parrot</li>
      </ul> 
    </li>
    <li class="item">
      <h2>Products</h2>
      <ul>
        <li>Bread</li>
        <li>Prasley</li>
        <li>Cheese</li>
      </ul>
    </li>
    <li class="item">
      <h2>Technologies</h2>
      <ul>
        <li>HTML</li>
        <li>CSS</li>
        <li>JavaScript</li>
        <li>React</li>
        <li>Node.js</li>
      </ul>
    </li>
  </ul>

Напиши скрипт, який:

  1. 1. Порахує і виведе в консоль кількість категорій в ulcategories, тобто елементів li.item.
  2. 2. Для кожного элемента li.item у списку ul#categories, знайде і виведе в консоль текст заголовку елемента (тегу <h2>) і кількість елементів в категорії (усіх <li>, вкладених в нього).

Для виконання цього завдання потрібно використати метод forEach() і властивості навігації по DOM.
В результаті, в консолі будуть виведені наступні повідомлення.

Логіка рішення

  • Отримати посилання.all на елемент .item
  • Вивести в консоль кількість li.item (елемент.lenght)
  • Перебрати в forEach() масив
  • В циклі законсолити firstElementChild.textContent
  • В циклі законсолити lastElementChild.children.length

Task 2

Завдання

HTML містить порожній список ul#ingredients.
<ul id="ingredients"></ul>

JavaScript містить масив рядків.

const ingredients = [ 
  "Potatoes", 
  "Mushrooms", 
  "Garlic",
  "Tomatos",
  "Herbs",
  "Condiments", 
];

Напиши скрипт, який для кожного елемента масиву ingredients:

  1. 1. Створить окремий елемент <li>. Обов'язково використовуй метод document.createElement().
  2. 2. Додасть назву інгредієнта як його текстовий вміст.
  3. 3. Додасть елементу клас item.
  4. 4. Після чого, вставить усі <li> за одну операцію у список ul#ingredients.

Логіка рішення

  • Взяти посилання на порожній список ul#ingredients.
  • Мепаємо масив ingredients. В мепі створюємо за допомогою методу document.createElement() лішки
  • Додаємо до них textContent і клас і повертаємо лі
  • Результати мепу розпилюємо в посилання .append()
